kareem abdul jabbar

Game Code Arduino

DESCRIPTION

"Kareem Abdul Jabbar" is a playful and intentionally silly title for a game created for my CART 253 final project. The game features three different characters across three distinct modes. In each mode, the player must dodge various objects being thrown at them, including bullets, rolls of toilet paper, and spaghetti tossed by an angry chef. During the bullet-dodging mode, players also have the ability to switch sides using the left arrow key, adding another layer of movement and strategy. The game can be played using a joystick connected through an Arduino setup, which I learned to implement with the help of Ms. Sabine. Visually and conceptually, the project draws inspiration from old school television wrestling and classic video games, combining retro elements to evoke a sense of nostalgia while playing. One of the most challenging parts of developing the game was designing the sprite sheets for all the characters, as it was my first time creating and working with custom sprite animations.
